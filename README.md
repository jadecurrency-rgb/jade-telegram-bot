# Jade Currency

Used JADE token real time balance to calculate voting totals of top trending BNB projects on binance smart chain.

## Technology Stack

- **Blockchain**: BNB Smart Chain + EVM-compatible chains  
- **Smart Contracts**: Solidity ^0.8.20.
- **Frontend**: [React / Vue / Next.js] + [ethers.js / web3.js / wagmi]  
- **Development**: [Hardhat / Foundry / Truffle], OpenZeppelin libraries
